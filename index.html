How To Use

What is this Mod?

Bonnie's Source Engine adds tool blocks, logic blocks, game blocks, and utility blocks for map creators in Minecraft Bedrock. This encyclopedia mirrors your in-game block workflow in one place.

Core UI Flow

Every block follows a 3-part editing pattern: Class Info, Outputs, and Inputs. Name + Start Disabled are global fields used by all block types.

Quick Tips

Use search to find blocks quickly, open any card for field-level details, and check output/input tabs when wiring larger systems. Conditions and examples in this app are based on your draft docs.
